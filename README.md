# 🛍 Retail Sales Analysis (SQL + Power BI)

## 📌 Project Overview
This project focuses on analyzing retail sales data using SQL and visualizing insights using Power BI.

The dataset used in this project was created manually using SQL INSERT statements. This project demonstrates end-to-end data analysis including database design, data creation, query execution, and dashboard development.

---

## 🧱 Database Structure
The project consists of three main tables:

- categories  
- products  
- sales  

These tables are connected using primary and foreign key relationships.

---

## 🛠 Tools & Technologies
- MySQL  
- Power BI  

---

## 📊 Key Features
- Created relational database using SQL  
- Inserted and managed data manually  
- Performed data analysis using SQL queries  
- Built interactive Power BI dashboard  
- Implemented KPI metrics for business insights  

---

## 📈 Dashboard Insights
- Electronics category generates the highest revenue  
- Laptop is the top-performing product  
- Monthly revenue shows a steady increase  
- Identified low stock products for inventory management  
- Average Order Value highlights customer purchasing behavior  

---

## 📊 Dashboard Features
- Total Revenue, Orders, Products, Quantity, Avg Order Value KPIs  
- Monthly Revenue Trend  
- Revenue by Category  
- Top Performing Products  
- Low Stock Alert (Inventory)  
- Interactive filtering using slicers  

---

## 📂 Dataset Information
The dataset was not taken from any external source.

It was manually created using SQL INSERT statements, which helped in understanding:
- Data structure  
- Table relationships  
- Real-world data simulation  

---

## ▶ How to Run the Project
1. Run the SQL file to create database and tables  
2. Insert data using the provided queries  
3. Connect Power BI to MySQL database  
4. Load the tables into Power BI  
5. View and interact with the dashboard
